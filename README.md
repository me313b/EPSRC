# EPSRC Proposal — Per-Slot Machine Drives

Working repository for the EPSRC Standard Grant proposal
"Foundations of Monolithically Integrated Per-Slot Machine Drives for Reconfigurable Aerospace Propulsion"
(PI: Dr Mehdi Baghdadi, UCL APL).

- **Start here:** `CLAUDE.md` (operating manual — Claude Code reads it automatically) and `SETUP.md`.
- **Brief:** `docs/` (concept & numbers · EPSRC rules · strategy · evidence · references · outstanding items).
- **Deliverables:** `proposal/` (one file per section; Status header governs the lock protocol).
- **Drafted prose:** `library/` (current openings + full archive).
- **Primary sources:** `sources/` (papers, theses, ATI strategy).
